Описание Rat Expert

Проект представляет собой веб-сервис, позволяющий любому человеку создавать и изменять свои экспертные системы, либо испытать системы других пользователей. Суть экспертной системы состоит в том, чтобы подобрать наиболее подходящие для пользователя объекты, узнав как можно больше параметров о нём. Для создания использовался фреймворк Django

Основные сущности системы

  • Пользователь
  • Система
  • Вопрос
  • Ответ
  • Параметр
  • Атрибут
  • Значение атрибута
  • Правило

Принцип работы

  • Пользователь создает систему и заполняет данные о ней

Принцип работы

  • После этого система доступна для прохождения

Принцип работы

  • Данные о параметрах пользователя, приоритетах объектов хранятся в сессии у пользователя
  • Каждым ответом пользователь отправляет выбранный парметр на сервер
  • На сервере для уже известных параметров пользователя проверяются правила
  • Существует 2 вида правил:
    1. параметр-параметр
    2. параметр-атрибут

Принцип работы

  • После проверки правил происходит обновление приоритетов объектов по имеющимся у них атрибутам
  • Вопрос может быть пропущен, тогда будет задан другой вопрос (при его наличии) на тот же параметр
  • Опрос заканчивается, когда будут известны все заданные параметры для этой системы

Области применения

Данная оболочка может применяться для создания систем, которые не предполагают отсеивания результатов. Также в системах обязательно наличие правил. Система не предполагает наличие жесткой логики задавания вопросов. При наличии соответствующих правил "параметр - параметр", возможны случаи, когда для одного параметра будут установлены несколько значений, либо вопросы для некоторых параметров не будут заданы - предполагается, что эксперт понимает, какие правила он создает.
